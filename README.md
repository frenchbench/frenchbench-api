# frenchbench-api
FrenchBench API

## Install

```
npm install
```

## Configure

Copy and edit `.env.sample`

## Build

```
npm run build
```

## Run

Development mode, run code in `src`:

```
npm run start:dev
```

Production mode, run built code in `dist`:

```
npm run start
```

## Test

Run test cases in `tests` using Mocha and Chai.

```
npm run test
```
